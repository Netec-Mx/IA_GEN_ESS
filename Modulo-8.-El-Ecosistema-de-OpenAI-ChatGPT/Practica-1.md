# Práctica 1. Analizar datos con Advanced Data Analysis
## Objetivos
Aplicar capacidades avanzadas de análisis de datos para interpretar información compleja y transformarla en resultados claros, accionables y orientados a la toma de decisiones.

## Duración aproximada
- 15 minutos.

## Tabla de ayuda
Para que puedas replicar esta práctica, se recomienda tener una cuenta en la siguiente plataforma:

| Sitio web | Enlace |
| --- | --- | 
| ChatGPT | https://auth.openai.com/create-account | 


## Instrucciones 
Sigue los pasos a continuación para completar cada tarea que conforma la práctica.


## Contexto de la práctica
Formas parte del área de análisis en una organización que trabaja con información demográfica para la toma de decisiones estratégicas (expansión de mercado, políticas públicas, segmentación de clientes, etc.).

Trabajarás con un dataset basado en el Censo de Población y Vivienda 2020 de México, el cual contiene información relevante como:
- Distribución poblacional
- Entidades federativas
- Variables demográficas
- Indicadores de población

Debes analizar los datos con Advanced Data Analysis (ADA) para generar insights relevantes y finalmente transformar el análisis en un documento ejecutivo dentro de Canvas.

### Parte 1. Exploración inicial del dataset
En esta etapa comprenderás la estructura de los datos.

1. Descarga el archivo [censo_pob_viv_2020.xlsx](../images/M8/P1/censo_pob_viv_2020.xlsx)
2. Ingresa a ChatGPT y crea un nuevo chat
3. Carga el archivo censo_pob_viv_2020.xlsx e integra el siguiente prompt:

```text
Explora este dataset y describe:

- Estructura de los datos
- Variables principales
- Tipo de información contenida
- Calidad de los datos (errores, valores nulos, inconsistencias)

Responde como analista.
```

4. Analiza la respuesta:
- ¿Comprende correctamente las variables?
- ¿Detecta problemas en los datos?
- ¿Identifica posibles relaciones?

### Parte 2. Análisis de datos
Ahora profundizarás en el comportamiento de la información.

1. Utiliza el siguiente prompt:

```text
Analiza el dataset e identifica:

- Tendencias clave en la población
- Patrones relevantes por estado o región
- Diferencias significativas entre entidades
- Posibles problemas o anomalías

Incluye ejemplos concretos basados en los datos.
```

2. Analiza la respuesta:
- ¿Identifica patrones reales?
- ¿Utiliza datos concretos?
- ¿El análisis es claro o superficial?

### Parte 3. Generación de KPIs
En esta etapa transformarás los datos en indicadores.

1. Utiliza el siguiente prompt:

```text
Genera KPIs clave a partir del dataset:

- Métricas principales de población
- Indicadores comparativos entre estados
- Identificación de concentraciones poblacionales
- Posibles riesgos o alertas

Incluye interpretación de cada KPI.
```

2. Analiza la respuesta:
- ¿Los KPIs son relevantes?
- ¿Están bien interpretados?
- ¿Podrían usarse en un reporte real?

### Parte 4. Generación de insights de negocio
Ahora convertirás el análisis en valor estratégico.

1. Utiliza el siguiente prompt:

```text
Actúa como consultor.
Convierte el análisis en:
- Insights estratégicos
- Problemas prioritarios
- Oportunidades de mejora o desarrollo

Enfoca la respuesta en toma de decisiones.
```

2. Analiza la respuesta:
- ¿Los insights son accionables?
- ¿Están conectados con los datos?
- ¿Aportan valor real?

### Parte 5. Creación del entregable en Canvas

1. Utiliza el siguiente prompt:

```text
Convierte el análisis en un entregable ejecutivo.

Incluye:
- Resumen ejecutivo
- KPIs clave
- Hallazgos principales
- Problemas detectados
- Recomendaciones

Formato claro pero aún en versión inicial en Canvas
```

Al entregar la respuesta, se debe abrir la sección en canvas. 
Deberías observar algo parecido a:

<img src="../images/M8/M8P1_1.png" alt="canvas"/>

Una de las ventajas de usar la versión en Canvas es que puedes editar directamente el contenido y después descargar el archivo creado o puedes compartirlo.
Da clic en Editar y explora todo lo que puedes hacer.

2. Analiza la respuesta:
- ¿Es claro para un directivo?
- ¿Está bien estructurado?
- ¿Es demasiado técnico?

### Parte 6. Refinar claridad

1. Para ajustar la información, utiliza el siguiente prompt:

```text
Haz el documento más claro y fácil de entender para un directivo.

Reduce complejidad innecesaria.
- Prioriza lo importante
- Elimina lo secundario
- Enfoca en decisiones

Convierte las recomendaciones en acciones concretas, definiendo:
- Qué hacer
- Quién debería hacerlo
- Impacto esperado
```

2. Solicita que se genere la versión final del documento, de manera que sea claro, breve, con un tono ejecutivo y que la información se encuentre lista para la toma de decisiones.


### Reflexión
- ¿Qué diferencia hay entre analizar datos y comunicar resultados?
- ¿Qué tan importante es estructurar la información para un directivo?
- ¿Cómo cambió el documento a lo largo del refinamiento?
- ¿Qué valor aporta el uso de Canvas en este proceso?
- ¿Qué errores iniciales detectaste en el análisis?

### Resultado esperado
Al finalizar la práctica, el participante será capaz de:
- Utilizar Advanced Data Analysis (ADA) para explorar datasets reales.
- Identificar patrones y tendencias en datos extensos y complejos.
- Generar KPIs relevantes a partir de información estructurada.
- Convertir análisis en insights estratégicos.
- Construir un entregable ejecutivo en Canvas.
