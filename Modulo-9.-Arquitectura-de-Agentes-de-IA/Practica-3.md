# Práctica 3. Diseñar la estructura de un agente
## Objetivos
Definir los componentes clave de un agente de IA, incluyendo identidad, conocimiento y acciones, para construir soluciones funcionales y alineadas a objetivos organizacionales.

## Duración aproximada
- 10 minutos.

## Tabla de ayuda
Para que puedas replicar esta práctica no necesitas ninguna herramienta mas que tu propia capacidad de análisis.

## Instrucciones 
Sigue los pasos a continuación para completar cada tarea que conforma la práctica.


## Contexto de la práctica
Selecciona un proceso o área de tu interés. Por ejemplo:
- Atención a clientes
- Recursos humanos
- Soporte técnico
- Finanzas
- Ventas

A partir de ese contexto, diseñarás la estructura lógica de un agente.

### Parte 1. Definir el contexto

1. Define brevemente:

- Área o proceso elegido:
- Problema que quieres resolver:
- Usuario al que atenderá el agente:

2. Define la identidad del agente:

- Nombre del agente:
- Rol (qué hace o cuál es su área de experiencia):
- Tipo de usuario que atiende:
- Tono (formal, cercano, técnico, etc.):

3. Define reglas de comportamiento:

Define al menos 3 reglas claras:

- Qué puede hacer
- Qué no puede hacer
- Cómo debe responder

Es importante que definas límites y control.

4. Define el conocimiento del agente:

Define qué necesita saber el agente para funcionar correctamente. Por ejemplo:

- Procesos
- Políticas
- Información operativa

Pregúntate: ¿Qué información necesita para responder correctamente?

5. Define acciones del agente:

Define qué puede hacer el agente dentro del proceso. Por ejemplo: 

- Clasificar solicitudes
- Detectar urgencias
- Sugerir pasos
- Escalar casos

Pregúntate: ¿Solo responde o también toma decisiones?

6. Integra todo en este formato:

```text
AGENTE: [Nombre]

Contexto:
- Área:
- Problema:
- Usuario:

Identidad:
- ...

Reglas:
- ...

Conocimiento:
- ...

Acciones:
- ...
```

### Reflexión

- ¿El agente está claramente definido?
- ¿Podría operar sin supervisión constante?
- ¿Qué pasaría si sus reglas no están bien definidas?
- ¿Qué riesgo existe en su uso?

### Resultado esperado
El participante habrá definido:
- Un agente con propósito claro
- Con reglas bien delimitadas
- Con conocimiento adecuado
- Con acciones concretas
